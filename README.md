# Avalia-o (Avaliação)

> Nome do repositório quebrou por causa do `ç` na URL — o conteúdo está dentro da pasta `Avaliação/`.

Quatro exercícios de programação web entregues como avaliação de uma matéria:

- `exercicio1.html` — JavaScript: calcula perímetro, área e diâmetro de um círculo a partir do raio
- `exercicio2.html` — JavaScript (front-end)
- `exercicio3.php` — PHP (back-end)
- `exercicio4.php` — PHP (back-end)

## Tecnologias

- HTML + JavaScript (exercícios 1 e 2)
- PHP (exercícios 3 e 4)

## Status

Avaliação acadêmica de 2020. Mantida como arquivo histórico — não recebe manutenção nem aceita PRs.
